# Kiran Android App Live

This repository contains the source code for the "Kiran" Android ecosystem, including the Main App (Receiver), Camera App (Broadcaster), and Remote Control App.

## Project Structure

- **1. Kiran Main App Live**: The receiver application that manages the AI pipeline and displays the video stream.
- **2. Kiran Camera App Live**: The broadcaster application that streams camera feed from the phone's camera.
- **3. Kiran Remote Control App**: Application for remote control operations.

## Features

- **High-Quality Streaming**: Real-time video streaming between devices.
- **AI-Powered Analysis**: Integrated YOLO v8 OBB and EfficientNet for object detection and SKU identification.
- **Multiple Camera Support**: Supports local camera, Wi-Fi stream, and USB camera feeds.
- **Remote Control**: Integrated remote input management.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/namemohit/Kiran-Android-app-live-
   ```
2. Open the project folders in Android Studio.
3. Build and run the applications on your Android devices.
